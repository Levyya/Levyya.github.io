Jsonnet 语法

发表于2023-01-11|更新于2023-01-12
|阅读量:

支持

  • 字符串格式化
  • 数组拼接，切片
  • python 表达式
  • 条件表达式
    • if condition then [] else []
  • 变量
    • :: 隐藏字段
    • local
    • self 当前对象
    • $ 根对象
  • 函数，与python函数类似，可以有默认值
  • 对象
    • super
    • + (覆盖组合)
    • +: (组合，不覆盖)
  • import

注释

  1. 多行注释: /* */
  2. 单行注释: //
