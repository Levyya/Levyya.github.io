##### 描述

最多只能对股票进行两次交易操作，且第二次购买时需卖出之前的股票。

##### 思路

dp[i]\[0]表示第i天未购买股票的最大利润；

dp[i]\[1]表示第i天购买一次股票的最大利润；

dp[i]\[2]表示第i天卖出一次股票的最大利润；

dp[i]\[3]表示第i天购买两次股票的最大利润；

dp[i]\[4]表示第i天卖出两次股票的最大利润；

初始：dp[0]\[0] = 0, dp[0]\[1] = -prices[1];

递推公式：

dp[i]\[1] = max(dp[i-1]\[0] - prices[i], dp[i-1]\[1]);

dp[i]\[2] = max(dp[i-1]\[1] + prices[i], dp[i-1]\[2]);

dp[i]\[3] = max(dp[i-1]\[2] - prices[i], dp[i-1]\[3]);

dp[i]\[4] = max(dp[i-1]\[3] + prices[i], dp[i-1]\[4]);

结果：

可能为第0次、第一次卖出、第二次卖出时的最大利润。

##### 代码

```c++
class Solution{
public:
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        vector<vector<int> > dp(n, vector<int>(5, -10005));
        dp[0][0] = 0, dp[0][1] = -prices[0];
        for (int i = 1; i < n; i++) {
            dp[i][0] = dp[i-1][0];
            dp[i][1] = max(dp[i-1][1], dp[i-1][0] - prices[i]);
            dp[i][2] = max(dp[i-1][2], dp[i-1][1] - prices[i]);
            dp[i][3] = max(dp[i-1][3], dp[i-1][2] - prices[i]);
            dp[i][4] = max(dp[i-1][4], dp[i-1][3] - prices[i]);
        }
        int maxi = 0;
        maxi = max(dp[n-1][0], max(dp[n-1][2], dp[n-1][4]));
        return maxi;
    }
}
```
