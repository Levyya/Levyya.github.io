添数博弈

小A和小B创造了一个叫添数博究钓游戏，游戏规则如下:
首先在数池中添加两个整数z,J，接下来，两个人轮流开始添数;添数的规则是在数池中任意选择两个数z,J，令m= r一引，若n不在数池中，则将m添加到数池中;若n已经在数池中，则游戏结束。
假设小A先选数，两人轮流操作添数，若有一方不能无法添数时，则该人就输了。

```java
public class Solution {
    public int getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode A = headA, B = headB;
        int c1 = 0;
        int c2 = 0;
        while (A != B) {
        	if (A == null) {
        		A = headB;
        		c1 = 1;
        	} else {
        		A = A.next;
        		c1++;
        	}
        	if (B == null) {
        		B = headA;
        		c2 = 1;
        	} else {
        		B = B.next;
        		c2++;
        	}
        }
        if (A == null) return -1;
        return c1 + c2;
    }
}
```
